About Kaptive

Kaptive is a tool for bacterial in silico serotyping. It takes one or more pre-assembled genomes and for each finds the best matching locus from a reference database. References for Klebsiella pneumoniae species complex, Acinetobacter baumannii and Vibrio parahaemolyticus are available in Kaptive's web interface.

The reference databases are curated by Kelly Wyres (Monash University, Australia) and Johanna Kenyon (Queensland University of Technology, Australia). Please contact us if you have an intact novel locus that you would like to be included.

The third-party database for Vibrio parahaemolyticus is maintained by Aldert Zomer.

Citations

If you use Kaptive's web interface please cite the paper below along with the reference for the relevant locus database(s).

Wick RR, Heinz E, Holt KE and Wyres KL 2018.
Kaptive Web: user-friendly capsule and lipopolysaccharide serotype prediction for Klebsiella genomes. Journal of Clinical Microbiology: 56(6). e00197-18
